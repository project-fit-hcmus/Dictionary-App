Hướng dẫn biên dịch và chạy chương trình 
> để chạy chương trình, chỉ cần mở termial và chỉ đường dẫn đến nơi chứa file run.jar 
> sau đó thực hiện chạy file run.jar bằng lệnh java -jar run.jar là chương trình sẽ tự động được chạy
> Sau khi chạy thì các file dữ liệu sẽ được lưu trong thư mục data. Có thể mở thư mục này để kiểm tra dữ liệu lịch sử tìm kiếm. File xml chứa danh sách từ yêu thích, file xml chứa danh sách từ điển tiếng Việt và từ điển tiếng Anh
