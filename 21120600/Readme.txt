Hướng dẫn biên dịch và chạy chương trình 
> để chạy chương trình, chỉ cần mở termial và chỉ đường dẫn đến nơi chứa file run.jar(nằm trong thư mục Release) 
> sau đó thực hiện chạy file run.jar bằng lệnh java -jar run.jar là chương trình sẽ tự động được chạy
> Sau khi chạy thì các file dữ liệu sẽ được lưu trong thư mục data. Có thể mở thư mục này để kiểm tra dữ liệu lịch sử tìm kiếm. File favorite_list.xml chứa danh sách từ yêu thích, file Anh_Viet.xml và Viet_Anh.xml chứa danh sách từ điển tiếng Anh và từ điển tiếng Việt. File history.txt để lưu lịch sử tìm kiếm. Tại đây một số lịch sử tìm kiếm từ trước khi bài được nộp vẫn lưu lại, có thể xóa file này để set lịch sử tìm kiếm lại rỗng từ lần chạy đầu.
